# 🌍 Air Quality Analysis Dashboard

## 📌 Overview
This project presents an interactive dashboard to analyze air quality data across various cities. It helps in understanding pollution levels and identifying the most polluted and clean cities.

## 🎯 Objectives
- Analyze AQI (Air Quality Index)
- Identify polluted cities
- Compare pollution components
- Provide interactive visualization

## ⚙️ Technologies Used
- Python
- Pandas
- Streamlit

## 📊 Features
- Average AQI calculation
- City-wise AQI filtering
- Top polluted cities
- Cleanest cities
- Pollution comparison chart

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py

## 📁 Dataset
Includes AQI and pollution metrics such as PM2.5, PM10, NO2, and CO.
